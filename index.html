Qu'est-ce que github ?

La plateforme github permet aux développeurs de stocker et également
de partager, publiquement ou pas le code qu'ils créent. Le site héberge
actuellement plus de 80 millions de projets, ils peuvent s'agirent de logiciels
de sites Web, d'applications mobiles ou tous autres types de logiciels informatique
avec n'importe quel language de programmation utilisé.

Le site est un espace collaboratif, cela permet a chaques utilisateurs de
contribuer aux projets mis en ligne en opensource, ils ont la possibilité
de proposer des modifications. Le succès de GitHub repose sur la facilité
de transmettre des informations sans que les utilisateurs ne se dérangent
mutuellement en apportant des modifications du programme en même temps.

Les modifications apporter au code sont stockée sur GitHub, il nous est
possible de suivre pas à pas chaque étape du développement d'un projet.
Nous pouvons trouver des espaces de discussion qui permettent à tous développeurs
d'échanger sur chaque programme et contribuer sur un même projet.
