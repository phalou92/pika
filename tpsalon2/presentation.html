Présentation des branches github


Les branches github c'est quoi ?

Les branches sur github représentent des versions alternatives du projet principal. Ce qui permet de développer de nouvelles fonctionnalités et les tester sans prendre le risque de tout casser.
Grâce a ce fonctionnement en branches n’importe qui peut proposer une amélioration qui elle même peut intégré au projet principal si le propriétaire en décide.
